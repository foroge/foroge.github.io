Учебный проект, созданный в рамках изучения **HTML**, **CSS** и **Bootstrap**.

Проект представляет собой многостраничный адаптивный сайт интернет-магазина компьютерной техники, опубликованный с помощью **GitHub Pages**. В ходе разработки были изучены основы верстки, построения структуры веб-страниц и работы с компонентами Bootstrap.

---

## Демо

**Сайт:** https://foroge.github.io

---

## Реализованная функциональность

- адаптивная верстка с использованием Bootstrap;
- многостраничная структура сайта;
- навигационное меню;
- карточки товаров;
- карусель изображений;
- адаптивная сетка Bootstrap;
- использование фонового видео;
- работа со статическими изображениями и иконками;
- подключение пользовательских CSS-стилей;
- адаптация интерфейса под различные размеры экрана.

---

## Структура сайта

- Главная страница
- Каталог товаров
- Навигационное меню
- Карточки товаров
- Контактная информация
- Footer
